
## Summary

* This is an implementation of the linear search algorithm in groovy.

## Environment setup

* jdk-1.8

## Compile and test

* To compile the code and the tests: 
```
./gradlew compileGroovy compileTestGroovy
```

* To run the tests : 
```
./gradlew test
```
